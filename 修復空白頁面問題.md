# 🔧 修復空白頁面問題

## ❌ 問題

網站顯示一片空白，沒有任何內容。

## 🔍 原因

GitHub Pages 部署在子目錄 `/student-marketplace/`，但 Vite 的 base path 設定錯誤。

## ✅ 解決方法

我已經修改了 `client/vite.config.js`，將 base path 改為 `/student-marketplace/`。

## 📤 重新部署步驟

### 步驟 1：提交修改

在 CMD 執行（專案根目錄）：

```bash
cd C:\test
git add .
git commit -m "Fix base path for GitHub Pages subdirectory"
git push
```

### 步驟 2：等待自動部署

- GitHub Actions 會自動偵測並重新部署
- 等待 2-3 分鐘

### 步驟 3：清除瀏覽器快取

1. 按 `Ctrl + Shift + R`（強制重新整理）
2. 或按 `F12` 開啟開發者工具
3. 右鍵點擊重新整理按鈕
4. 選擇「清除快取並強制重新整理」

### 步驟 4：測試網站

訪問：
```
https://yourname1218.github.io/student-marketplace/
```

應該會看到網站內容了！

## 🔍 如果還是空白

### 檢查瀏覽器控制台

1. 按 `F12` 開啟開發者工具
2. 點擊「Console」標籤
3. 查看是否有紅色錯誤訊息
4. 把錯誤訊息告訴我

### 常見錯誤

**錯誤 1：404 找不到檔案**
- 檢查 base path 是否正確設定

**錯誤 2：CORS 錯誤**
- 檢查後端 CORS 設定
- 確認 `server/server.js` 中的 GitHub Pages 網址正確

**錯誤 3：API 請求失敗**
- 檢查 `VITE_API_URL` Secret 是否正確設定
- 確認 Render 後端正常運行
