# 🚀 快速開始指南

## 📦 安裝步驟

### 1. 安裝 Node.js
如果還沒安裝，請先從 https://nodejs.org/ 下載並安裝 Node.js LTS 版本

### 2. 安裝專案依賴

在專案根目錄執行（使用 CMD 或 PowerShell）：

```bash
npm run install:all
```

這會自動安裝前端和後端所有需要的套件。

### 3. 建立範例資料（重要！）

為了讓網站有漂亮的範例商品，請執行：

```bash
cd server
npm run seed
```

這會建立：
- ✅ 7 位測試使用者
- ✅ 60 個精美的範例商品（每個類別10個）
  - 📚 教科書
  - 💻 3C產品
  - ✏️ 文具用品
  - 🏠 生活用品
  - ⚽ 運動器材
  - 🎁 其他

### 4. 啟動專案

回到專案根目錄：

```bash
cd ..
npm run dev
```

### 5. 開啟瀏覽器

前往 http://localhost:3000 開始使用！

## 🐛 常見問題

### Q: 執行 npm 指令時出現錯誤？
A: 確保已經安裝 Node.js 並重新啟動終端機

### Q: 網站沒有顯示商品？
A: 請執行 `cd server` 然後 `npm run seed` 建立範例資料

### Q: PowerShell 執行原則錯誤？
A: 建議使用 CMD 而非 PowerShell，或以系統管理員身分修改執行原則

### Q: 可以重新建立範例資料嗎？
A: 可以！再次執行 `npm run seed` 即可，舊資料會被清除
